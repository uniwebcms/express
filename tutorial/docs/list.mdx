# List

A component that can render a list of items.

<Gallery images={['/img/list/outcome_1.png','/img/list/outcome_2.png','/img/list/outcome_3.png','/img/list/outcome_4.png']} sizes={[[1245, 951],[1606, 544],[1540, 953],[1240, 687]]}/>

#### List item data source: 
- The pre-selected profiles in a `List` profile.
- The downloadable assets of the main profile.
- The items from a docufolio.
- The linked profiles to the main profile.
